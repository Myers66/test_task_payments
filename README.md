# Тестовое задание: визуализация платежей

**Разработчик:** Иван Евдокимов  
**Дата выполнения:** май 2026

## Описание

Одностраничное веб-приложение на Django для визуализации платежей.  
График зависимости суммы платежа от даты (линейный) + возможность переключения на гистограмму, показывающую общую сумму платежей по каждому клиенту (имя + фамилия).

## Технологии

- Python 3.11
- Django 4.2
- PostgreSQL 15
- Chart.js 4.4
- Bootstrap 5
